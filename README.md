# InternshipProject — SkillCraft Technology

This repository contains the 4 tasks completed as part of the SkillCraft Technology internship program.

## 📁 Repository Structure

```
InternshipProject/
├── tempconvert.py           # Task 01 - Temperature Converter
├── numguess.html             # Task 02 - Number Guessing Game
├── sudukosolve.html          # Task 03 - Sudoku Solver
├── mern-product-scraper/     # Task 04 - E-commerce Product Scraper (MERN stack)
└── README.md
```

---

## Task 01 — Temperature Converter
**File:** `tempconvert.py`

A program that converts temperature values between Celsius, Fahrenheit, and Kelvin scales.

**Run it:**
```bash
python3 tempconvert.py
```

---

## Task 02 — Number Guessing Game
**File:** `numguess.html`

Generates a random number and challenges the user to guess it, prompting for input and comparing it to the generated number until the correct guess is made.

**Run it:**
Open `numguess.html` in any web browser.

---

## Task 03 — Sudoku Solver
**File:** `sudukosolve.html`

Solves Sudoku puzzles automatically — takes an input grid representing an unsolved puzzle and fills in the missing numbers using a solving algorithm.

**Run it:**
Open `sudukosolve.html` in any web browser.

---

## Task 04 — E-commerce Product Scraper
**Folder:** `mern-product-scraper/`

Extracts product information — names, prices, and ratings — from an online e-commerce website and stores the data in a structured format (CSV).

Built using the MERN stack (MongoDB, Express, React, Node.js).

**Run it:**
See setup instructions inside the `mern-product-scraper/` folder.

---

## 🛠️ Tech Stack

- Python (Task 01)
- HTML / JavaScript (Tasks 02 & 03)
- MERN stack — MongoDB, Express, React, Node.js (Task 04)
